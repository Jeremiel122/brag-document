## 📋 What is brag-document?

brag-document helps you keep track of your career achievements and work history in a simple notebook. Instead of just writing plain text, it saves your notes in an easy-to-read format. It organizes your career info so you can use it later with smarter tools if you want.

This project builds on the idea of a "Brag Document," which is a way to note your wins and contributions at work continuously. This helps you remember your progress and what you’ve done. The brag-document uses Markdown files inside a folder, which makes your notes easy to structure and read.

## 📝 How to Use brag-document

1. Open brag-document by clicking the shortcut.

2. The app shows a simple window where you can add your career notes.

3. Use the buttons to create new records. Each record can be about a project, achievement, or learning.

4. Write your notes in plain text. You can use Markdown formatting if you want bold or lists.

5. Save your updates to keep your brag document organized.

6. You can browse past entries any time to review what you have done.

- Focus on what you did and the results you achieved
